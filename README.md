# 原神多账号自动化一条龙 v5.1 —— 使用说明

## 概述

本工具可以自动切换多个原神账号，逐号调用 BetterGI 完成每日委托、清体力等自动化操作。支持直接启动/胡桃启动两种方式，内置定时计划，勾选开机自启后可以做到无人值守。

**必备依赖**：BetterGI、原神游戏。胡桃工具箱为可选。

---

## 第一步：安装依赖

### 1. BetterGI

BetterGI 是真正执行游戏内自动化操作的软件（战斗、委托、领取奖励等）。

- 下载地址：https://github.com/babalae/better-genshin-impact
- 下载最新 Release 的压缩包，解压到任意目录（不需要安装）。
- 打开 BetterGI，在「自动任务」→「一条龙」页面配置好你的任务流程。（不会配去 B 站搜"BetterGI 一条龙教程"）

### 2. 原神

确保原神已安装，且启动方式为**窗口化**（设置 → 图像 → 显示模式 → 窗口化）。

### 3. 胡桃工具箱（可选）

如果你需要胡桃切号功能：

- 下载地址：https://github.com/DGP-Studio/Snap.Hutao
- MSIX 版（从 Microsoft Store 安装）需在设置中填写 AppID；普通解压版无需 AppID。

### 4. Tesseract OCR（已内置，无需安装）

压缩包中已自带 `tesseract-ocr` 文件夹，包含中英文语言包。首次使用时在设置里将「Tesseract OCR 目录」指向该文件夹即可，无需自行下载安装。

---

## 第二步：文件部署

将 `GenshinAutoTool_NoAccounts.exe` 放到一个英文路径的文件夹（如 `D:\GenshinTool`），不要放在中文或含空格的路径下。

`config.json` 首次运行会自动生成，无需手动创建。

---

## 第三步：首次设置

双击 exe 启动程序后，点击**标题栏右侧的「设置」**按钮，打开设置窗口。

| 序号 | 项目 | 说明 |
|:---:|------|------|
| 1 | BetterGI 可执行文件 | 点「浏览」选择 BetterGI 目录下的 `BetterGI.exe` |
| 2 | BetterGI 配置文件 | 点「浏览」选择 BetterGI 的 `User/config.json` |
| 3 | 原神可执行文件 | 点「浏览」选择原神目录下的 `YuanShen.exe`（国际服选 `GenshinImpact.exe`） |
| 4 | 原神进程名 | 默认 `YuanShen.exe`，国际服改 `GenshinImpact.exe` |
| 5 | 胡桃工具箱路径 | （可选）选择胡桃的 `Snap.Hutao.Remastered.FullTrust.exe` |
| 6 | 胡桃 AppID | （可选）仅 MSIX 版胡桃需要。判断方法：右键胡桃快捷方式 → 属性，路径中含有 `WindowsApps` 即为 MSIX 版。在 PowerShell 中执行 `Get-AppxPackage *Hutao*` 获取 AppID。普通解压版留空。 |
| 7 | 一条龙超时时间 | 单个账号最大等待时间（秒），默认 600。账号多或任务重的可以改大。 |
| 8 | 全局停止快捷键 | 点「点击设置」后按下键盘组合键（如 F8、Ctrl+Shift+Q），支持修饰键+单键 |
| 9 | 启动时自动最小化 | 勾选后，程序启动时自动缩到系统托盘，不挡屏幕 |
| 10 | Tesseract OCR 目录 | 点「浏览」选择解压后自带的 `tesseract-ocr` 文件夹 |
| 11 | UID 识别方式 | Tesseract OCR 多阈值投票法（推荐）；BetterGI 调度组（更准，需先配置脚本） |

填写完毕后点「保存」。

---

## 第四步：添加账号

回到主界面，在**左侧面板**底部点「+ 添加账号」，弹出账号编辑窗口：

| 字段 | 说明 |
|------|------|
| **账号名称** | 任意起名，如"大号""小号""亚服号" |
| **启动方式** | 「直接启动(BetterGI)」——绝大多数人用这个；「胡桃启动」——通过胡桃工具箱切换账号后启动 |
| **BetterGI 一条龙配置** | 下拉选择 BetterGI OneDragon 目录下的配置名。如果只有一个配置，自动选中 |
| **调度器配置组** | UID 不匹配时执行的 BetterGI ScriptGroup。通常配合切号脚本使用，普通用户留空即可 |
| **胡桃中的账号名称** | 仅「胡桃启动」模式需要填写，必须与胡桃工具箱中保存的账号名完全一致 |
| **游戏UID** | 9 位数字，用于切号后验证是否登入了正确账号 |
| **任务完成后关闭软件** | 三个独立复选框：关闭游戏 / 关闭 BetterGI / 关闭胡桃。默认全勾，可根据需要取消某些项 |

添加后账号会出现在左侧列表中，勾选（打 √）需要执行的账号。

**其他操作**：

- 点击行内的「编辑」可修改账号信息
- 选中一个账号后点「删除」可移除
- **拖拽排序**：按住账号行拖动可调整执行顺序（顺序即执行顺序）
- 底部「全选」可一键勾选全部账号

---

## 第五步：开始执行

**启动前确认**：

- 原神、BetterGI 均未运行（程序会自动启动它们）
- 游戏已设为窗口化模式
- 一条龙执行期间不要动鼠标键盘

点击右下角**「▶ 开始」**按钮，程序将按以下流程逐号执行：

1. 启动原神
2. 识别当前 UID，验证账号是否正确
3. 若 UID 不匹配，执行调度器配置组（如设置了切号脚本）
4. 启动 BetterGI 并执行一条龙任务
5. 监控 BetterGI 日志，等待「任务结束」完成信号
6. 任务完成后再等 10 秒确保所有操作收尾
7. 根据设置关闭游戏 / BetterGI / 胡桃
8. 切换到下一个账号，重复以上流程

右侧日志区域会实时显示进度，进度条同步更新。过程中可按「停止」按钮或全局快捷键强制中断。

---

## 定时计划

点击标题栏**「定时计划」**，打开定时器管理窗口。

### 添加任务

1. **执行模式**：每天 / 每周 / 一次性 / 指定日期
2. **时间**：点击快捷时间按钮（04:00 / 08:00 / 18:00 / 22:00）快速设置，也可手动选择时/分
3. **日期/星期**：根据模式显示不同选择器
   - 每天：无需选择
   - 每周：勾选周一至周日
   - 一次性：选择一个具体日期
   - 指定日期：多选多个日期
4. **账号**：勾选本次定时执行的账号
5. **调度组**：可选的共用调度组（ScriptGroup），适用于需要统一切号脚本的场景
6. 点击「添加任务」

### 管理任务

- 已设任务列表显示所有计划，可勾选启/禁用
- 「全选」一键选中全部任务
- 「删除选中」移除选中的任务
- 「开机自启动」勾选后写入注册表，电脑重启后自动启动程序并运行定时器
- 「启动定时器」开始后台轮询，每分钟检查一次时间。**启动后窗口自动最小化到托盘**
- 「停止定时器」停止后台轮询

### 定时器行为

- 每分钟轮询，到点自动触发一条龙
- 同一任务在一天内不会重复执行
- 如果当前正在执行中，新触发的任务自动跳过
- 每种模式互不干扰，可混合设置

---

## 托盘与窗口管理

### 托盘行为

- 关闭窗口（点 ×）时**最小化到系统托盘**（通知区域），非任务栏
- 右键托盘图标：「显示窗口」恢复界面 / 「退出」彻底关闭程序
- 定时器启动后自动最小化到托盘，不占用桌面空间

### 置顶

点击标题栏 **📌** 按钮可将窗口置顶（始终在最前），再次点击取消置顶。

## UID 识别详解

### 方式一：Tesseract OCR（默认）

使用多阈值投票法识别游戏画面中的 UID。需在设置中配置 Tesseract OCR 目录（指向自带的 `tesseract-ocr` 文件夹）。

### 方式二：BetterGI 调度组（更准确）

利用 BetterGI 的脚本能力识别 UID，适合 OCR 识别不稳定时使用。

配置方法：
1. 将压缩包中 `BetterGI-UID识别脚本` 文件夹放到 BetterGI 的 `User/JsScript/` 目录
2. 在 BetterGI 中新建一个调度组，把 UID 识别脚本加入该组
3. 在程序设置中 UID 识别方式选「BetterGI 调度组」，下拉选择刚才创建的调度组名称

---

## 调度器切号说明

定时计划中的「调度组」用于执行 BetterGI ScriptGroup 中的脚本（如切号脚本）。

BetterGI 官方脚本仓库：https://github.com/babalae/bettergi-scripts-list
- 推荐切号脚本：`repo/js/SwitchAccountMultipleMode`
- 按照脚本自带的说明安装配置后，将脚本加入 BetterGI 调度组
- 在程序账号设置中将「调度器配置组」指向该组即可

## 常见问题

**Q：中途怎么停下来？**

A：点「停止」按钮，或按设置中配置的全局快捷键（默认 Ctrl+Shift+Q）。

**Q：提示"未检测到 UID"？**

A：① 检查 Tesseract OCR 目录路径是否正确；② 确认游戏是窗口化运行；③ 游戏界面没有遮挡。如果 OCR 持续失败，可切换到「BetterGI 调度组」识别方式。

**Q：BetterGI 启动了但游戏里没反应？**

A：① 检查 BetterGI 中一条龙任务是否已正确配置；② 确认 BetterGI 配置文件路径填对了；③ 点击游戏窗口让它置顶。

**Q：账号切换失败？**

A：① 确认 BetterGI 配置文件路径正确；② 如果是胡桃启动模式，检查胡桃路径和账号名称是否与胡桃中完全一致；③ 检查 UID 是否正确填写。

**Q：程序能打开多个吗？**

A：不能。程序会单实例锁定，重复启动会弹出提示「程序已在运行中」并退出。

**Q：窗口关闭后怎么找回来？**

A：程序缩小到了系统托盘（通知区域右下角），右键托盘图标选「显示窗口」即可。只有右键托盘「退出」才会真正关闭程序。

**Q：进度条不走？**

A：进度条显示当前账号的执行进度（0/勾选数）。如果某个号卡住，检查 BetterGI 日志是否有报错，或调大超时时间。

**Q：定时器到时间不执行？**

A：确认定时器状态显示「定时器运行中」。如果当前有任务正在执行（手动或上次定时触发的），新的定时任务会跳过。

---

## 注意事项

- 游戏必须窗口化运行，全屏无法识别 UID
- 一条龙执行期间不要操作鼠标键盘，会干扰自动化流程
- 首次使用建议只勾选一个账号试运行，确认流程正常后再批量执行
- 电脑不要开启休眠/睡眠，避免定时任务无法触发
- 不同账号的 BetterGI 一条龙配置可以是不同的，每个账号独立选择
- 「调度器配置组」是 UID 不匹配时才触发的兜底机制，不是必填项，普通用户无需关心
