# 原神多账号辅助工具 v5.4

## 概述

本工具可以自动切换多个原神账号，逐号调用 BetterGI 完成每日委托、清体力等自动化操作。支持直接启动 / 胡桃启动两种方式，内置定时计划，勾选开机自启后可以做到无人值守。

**必备依赖**
- BetterGI（执行游戏内自动化）
- 原神游戏（窗口化运行）
- 胡桃工具箱（仅切号功能需要，可选）
- Tesseract OCR（压缩包已自带，无需单独安装）

---

## 第一步：安装依赖

### 1. BetterGI

BetterGI 是真正执行游戏内自动化操作的软件（战斗、委托、领取奖励等）。

- 下载地址：https://github.com/babalae/better-genshin-impact
- 下载最新 Release 的压缩包，解压到任意目录（不需要安装）。
- 打开 BetterGI，在「自动任务」→「一条龙」页面配置好你的任务流程。（不会配去 B 站搜 "BetterGI 一条龙教程"）

### 2. 原神

确保原神已安装，且启动方式为窗口化：`设置 → 图像 → 显示模式 → 窗口化`。

### 3. 胡桃工具箱（可选）

如果你需要胡桃切号功能：

- 下载地址：https://github.com/DGP-Studio/Snap.Hutao
- MSIX 版（Microsoft Store 安装）需在设置中填写 AppID；普通解压版无需 AppID。

判断是否 MSIX 版的方法：右键胡桃快捷方式 → 属性，路径中含有 `WindowsApps` 即为 MSIX 版。在 PowerShell 中执行 `Get-AppxPackage *Hutao*` 获取 AppID。

### 4. Tesseract OCR（已内置，无需安装）

压缩包中已自带 `tesseract-ocr` 文件夹，包含中英文语言包。首次使用时在设置里将「Tesseract OCR 目录」指向该文件夹即可。

## 第二步：文件部署

1. 将压缩包中所有文件解压到同一个文件夹（如 `D:\GenshinTool`）。
2. 文件夹路径不要含中文或空格。
3. 确保以下文件/文件夹在同一目录下：

| 文件 | 说明 |
|------|------|
| `GenshinMultiAccountTool.exe` | 主程序 |
| `config.json` | 首次运行自动生成，不要手动编辑！ |
| `tesseract-ocr/` | OCR 引擎（**必须和 exe 同级**） |

> `tesseract-ocr/` 文件夹必须和 exe 放在同一目录，否则 UID 识别会失败。

## 第三步：首次设置

双击 `GenshinMultiAccountTool.exe` 启动程序（窗口标题为 `GenshinMultiAccountTool v5.4`），点击标题栏右侧的「设置」按钮。

**设置项共 14 项：**

| 序号 | 项目 | 说明 |
|:---:|------|------|
| 1 | BetterGI 可执行文件 | 点「浏览」选择 BetterGI 目录下的 `BetterGI.exe`。不知道在哪？桌面找 BetterGI 快捷方式 → 右键 →「打开文件所在位置」，里面就能找到。常见位置：`C:\Program Files\BetterGI\BetterGI.exe` 或 `D:\BetterGI\BetterGI.exe`。实在找不到就在文件夹里搜索 `BetterGI.exe`。 |
| 2 | BetterGI 配置文件 | 点「浏览」选择 BetterGI 的 `User\config.json`。找到 BetterGI 文件夹后，里面有个 `User` 子文件夹，进去选 `config.json` 即可。如果没看到 `User` 文件夹，说明还没运行过 BetterGI——打开 `BetterGI.exe` 一次（不进游戏也行），关掉后文件夹会自动生成。 |
| 3 | 原神可执行文件 | 点「浏览」选择原神目录下的 `YuanShen.exe`（国际服选 `GenshinImpact.exe`）。桌面找原神快捷方式 → 右键 →「打开文件所在位置」。如果快捷方式指向的是启动器，进去还有个 `Genshin Impact Game` 子文件夹，点进去就能看到。常见位置：`C:\Program Files\Genshin Impact\Genshin Impact Game\YuanShen.exe`。 |
| 4 | 原神进程名 | 默认 `YuanShen.exe`，国际服改 `GenshinImpact.exe`。用来判断游戏是否在运行，写错检测不到。怎么判断国服还是国际服：用国内手机号登录的是国服，用邮箱和国外账号登录的是国际服。 |
| 5 | 胡桃工具箱路径（可选） | 仅胡桃切号时需要。选择胡桃的 `Snap.Hutao.exe`。桌面找胡桃快捷方式 → 右键 →「打开文件所在位置」即可定位。常见位置：`C:\Users\你的用户名\AppData\Local\Hutao\Snap.Hutao.exe`。 |
| 6 | 胡桃 AppID（可选） | 仅 MSIX 版胡桃需要。普通解压版留空。判断和获取方法见上文「第一步 > 胡桃工具箱」。 |
| 7 | 一条龙超时时间 | 单个账号最大等待时间（秒），默认 7200（2 小时）。账号多或任务重的可以改大，超时未完成会跳过当前号。 |
| 8 | 全局停止快捷键 | 点「点击设置」后按下键盘组合键（如 `F8`、`Ctrl+Shift+Q`），支持修饰键 + 单键。一条龙执行中按下此键可强制停止。 |
| 9 | 全局暂停/继续快捷键 | 点「点击设置」后按下键盘组合键（默认 `Ctrl+Shift+P`）。一条龙执行中按下可暂停/继续。 |
| 10 | 启动时自动最小化 | 勾选后，程序启动时自动缩到系统托盘，不挡屏幕。配合定时器的「一键启动全部」使用效果最佳。 |
| 11 | 关闭窗口时最小化到系统托盘 | 勾选后，点窗口右上角 X 不会退出程序，只会缩到右下角托盘图标。要完全退出需右键托盘图标选「退出」。 |
| 12 | 所有任务完成后自动关机 | 勾选后，全部账号执行完毕后弹出 60 秒倒计时，届时自动关机。倒计时期间可点击按钮取消。 |
| 13 | Tesseract OCR 目录 | 点「浏览」选择解压后自带的 `tesseract-ocr` 文件夹（和 exe 同级）。如果自己装过 Tesseract，常见位置：`C:\Program Files\Tesseract-OCR`，找不到就搜索 `tesseract.exe`。 |
| 14 | UID 识别方式 | ① Tesseract OCR 多阈值投票法（推荐）——需配好 Tesseract OCR 目录，优点不依赖 BetterGI，缺点必须窗口化、画面不能被遮挡。② BetterGI 调度组（更准）——BetterGI 内部直接读 UID，不需要 Tesseract，准确度更高。选这个后在下拉框里选一个 BetterGI 调度组即可。 |

填写完毕后点「保存」。

## 第四步：添加账号

回到主界面，在左侧面板底部点「**+ 添加账号**」，弹出账号编辑窗口。

| 字段 | 说明 |
|------|------|
| 账号名称 | 任意起名，如 "大号""小号""亚服号" |
| 启动方式 | **直接启动(BetterGI)**——绝大多数人用这个，工具直接帮你启动原神然后调 BetterGI；**胡桃启动**——通过胡桃工具箱切换账号后启动。区别：直接启动模式只需原神 + BetterGI；胡桃启动模式额外依赖胡桃工具箱做切号。 |
| BetterGI 一条龙配置 | 下拉选择 BetterGI OneDragon 目录下的配置名。如果只有一个配置，自动选中。如果下拉为空，说明 BetterGI 尚未配置过一条龙任务。 |
| 调度器配置组 | UID 不匹配时执行的 BetterGI ScriptGroup。通常配合切号脚本使用，普通用户留空即可。 |
| 胡桃中的账号名称 | 仅「胡桃启动」模式需要填写，必须与胡桃工具箱中保存的账号名完全一致。 |
| 游戏UID | 9 位数字，用于切号后验证是否登入了正确账号。 |
| 任务完成后关闭软件 | 三个独立复选框：关闭游戏 / 关闭BetterGI / 关闭胡桃。默认全勾，可根据需要取消某些项。 |

添加后账号会出现在左侧列表中，勾选（打 √）需要执行的账号。

**其他操作：**
- 点击行内的「编辑」可修改账号信息
- 选中一个账号后点「删除」可移除
- 拖拽排序：按住账号行拖动可调整执行顺序（顺序即执行顺序）
- 底部「全选」可一键勾选/取消全部账号

## 第五步：开始执行

**启动前确认：**
- 原神、BetterGI 均未运行（程序会自动启动它们）
- 游戏已设为窗口化模式
- 一条龙执行期间不要动鼠标键盘

点击右下角「▶ 开始」按钮。

**按钮状态变化：**

| 阶段 | 开始 | 暂停 | 停止 |
|------|:---:|:---:|:---:|
| 开始前 | 蓝色可点 | 灰色不可点 | 灰色不可点 |
| 运行中 | 灰色 | 橙色可点 | 红色可点 |
| 暂停时 | 灰色 | 绿色「▶ 继续」 | 红色可点 |
| 停止后 | 蓝色可点 | 灰色不可点 | 灰色不可点 |

**执行流程（逐号自动完成）：**
1. 启动原神
2. 识别当前 UID，验证账号是否正确
3. 若 UID 不匹配，执行调度器配置组（如设置了切号脚本）
4. 启动 BetterGI 并执行一条龙任务
5. 监控 BetterGI 日志，等待「任务结束」完成信号
6. 任务完成后再等 10 秒确保所有操作收尾
7. 根据设置关闭游戏 / BetterGI / 胡桃
8. 切换到下一个账号，重复以上流程

> 如果开启了「所有任务完成后自动关机」，全部账号完成后底部会弹出红色的「✕ 取消关机」按钮，同时状态栏显示 60 秒倒计时。点击该按钮可取消关机。

右侧日志区域会实时显示进度，底部进度条同步更新。

## 快捷键（热键）

程序提供两套全局快捷键，无论当前窗口焦点在哪里都能响应：

| 快捷键 | 默认值 | 功能 |
|--------|--------|------|
| 全局停止 | `Ctrl+Shift+Q` | 强制停止当前一条龙任务 |
| 全局暂停/继续 | `Ctrl+Shift+P` | 暂停或继续当前任务 |

- 快捷键可在「设置」中自定义，支持单键（`F8`）或组合键（`Ctrl+Shift+Q`）。
- 程序启动后热键自动注册，关闭程序后自动注销。
- 底部状态栏会显示当前已注册的快捷键。

## 系统托盘

程序运行后会出现在任务栏右下角托盘区域（tooltip 显示 `GenshinMultiAccountTool v5.4`）。

**左键点击托盘图标**：显示主窗口

**右键点击托盘图标**，弹出菜单：
- **显示窗口**——打开主界面
- **开始任务**——同主界面「▶ 开始」
- **暂停任务**——暂停当前一条龙
- **停止任务**——停止当前一条龙
- **取消自动关机**——取消倒计时自动关机（仅在倒计时期间生效）
- **定时器**——子菜单，展开后显示：
  - 每个定时任务的独立启用/停用勾选项
  - 一键启动全部 / 一键关闭全部
  - 有任务启用时，父菜单会显示勾选标记
- **设置**——打开设置窗口
- **退出**——彻底关闭程序

> 注意：右键菜单里的「退出」才是真正关闭程序；点窗口右上角 X 只是缩到托盘（前提是设置了「关闭窗口时最小化到系统托盘」）。

## 定时计划

点击标题栏「定时计划」，打开定时器管理窗口（与主窗口同款图标）。

### 添加任务

1. **执行模式**：每天 / 每周 / 一次性 / 指定日期（多选多个具体日期）
2. **时间**：点击快捷时间按钮（凌晨4点 / 早8点 / 午12点 / 晚8点）快速设置，也可手动选择时/分
3. **日期/星期**：根据模式显示不同选择器
   - 每天：无需选择日期
   - 每周：勾选周一至周日
   - 一次性：选择一个具体日期
   - 指定日期：多选多个具体日期（如选了 3号、10号、17号）
4. **账号**：勾选本次定时执行的账号
5. **调度组**：可选的共用调度组（ScriptGroup），适用于需要统一切号脚本的场景。与账号设置中的「调度器配置组」不同：此处是任务级别的统一调度组，执行前会先运行该组的脚本。
6. 点击「添加任务」

### 管理任务

窗口顶部栏提供：
- **一键启动全部**——一次性启动所有定时任务，定时器开始运作
- **停止定时器**——停止正在运行的定时器
- **开机自启**——勾选后电脑开机自动运行程序
- **完成自动关机**——全部任务结束后自动关机

「已设任务」列表中，每个任务行右侧有独立的启用/停用按钮，可以单独开关某个任务而不影响其他任务。顶部「全选」和「删除选中」按钮可批量管理。

右侧状态栏显示当前定时器运行状态。

## UID 识别详解

### 方式一：Tesseract OCR（默认）

使用多阈值投票法识别游戏画面中的 UID。需要 `tesseract-ocr` 文件夹与 exe 在同一目录，并在设置中将「Tesseract OCR 目录」指向该文件夹。

### 方式二：BetterGI 调度组（更准确）

利用 BetterGI 的脚本能力识别 UID，适合 OCR 识别不稳定时使用。

**配置方法：**
1. 将 BetterGI 的 UID 识别脚本放到 BetterGI 的 `User/JsScript/` 目录
2. 在 BetterGI 中新建一个调度组，把 UID 识别脚本加入该组
3. 在程序设置中 UID 识别方式选「BetterGI 调度组」，下拉选择刚才创建的调度组名称

---

## 调度器切号说明

定时计划中的「调度组」用于执行 BetterGI ScriptGroup 中的脚本（如切号脚本）。

BetterGI 官方脚本仓库：https://github.com/babalae/bettergi-scripts-list

推荐切号脚本：`repo/js/SwitchAccountMultipleMode`，按照脚本自带的说明安装配置后，将脚本加入 BetterGI 调度组，在程序账号设置中将「调度器配置组」指向该组即可。

## 常见问题

**Q：中途怎么停下来？**
A：点「停止」按钮，或按设置中配置的全局停止快捷键（默认 `Ctrl+Shift+Q`）。也可以右键托盘图标选「停止任务」。

**Q：怎么临时暂停？**
A：点「暂停」按钮或按 `Ctrl+Shift+P`，再点一次或再按一次快捷键继续。

**Q：提示"未检测到 UID"？**
A：① 检查 Tesseract OCR 目录路径是否正确；② 确认游戏是窗口化运行；③ 游戏界面没有遮挡；④ `tesseract-ocr` 文件夹必须和 exe 在同一目录。如果 OCR 持续失败，可切换到「BetterGI 调度组」识别方式。

**Q：BetterGI 启动了但游戏里没反应？**
A：① 检查 BetterGI 中一条龙任务是否已正确配置；② 确认 BetterGI 配置文件路径填对了；③ 点击游戏窗口让它置顶。

**Q：账号切换失败？**
A：① 确认 BetterGI 配置文件路径正确；② 如果是胡桃启动模式，检查胡桃路径和账号名称是否与胡桃中完全一致；③ 检查 UID 是否正确填写。

**Q：程序能打开多个吗？**
A：不能。程序会单实例锁定，重复启动会弹出提示「程序已在运行中」并退出。如果找不到已打开的窗口，右键托盘图标选「显示窗口」。

**Q：窗口关闭后怎么找回来？**
A：程序缩小到了系统托盘（通知区域右下角），双击托盘图标或右键选「显示窗口」即可。只有右键托盘「退出」才会真正关闭程序。

**Q：进度条不走？**
A：进度条显示当前账号的执行进度（0/勾选数）。如果某个号卡住，检查 BetterGI 日志是否有报错，或调大超时时间。

**Q：定时器到时间不执行？**
A：① 确认定时器状态显示「运行中」；② 电脑不能休眠或睡眠（系统设置 → 电源 → 改成"永不"）；③ 如果当前有任务正在执行（手动或上次定时触发的），新的定时任务会跳过；④ 可以先手动点「一键启动全部」试一次看是否能正常运行。

**Q：自动关机怎么取消？**
A：关机倒计时期间，主界面底部会出现红色「✕ 取消关机」按钮，点击即可取消。也可右键托盘图标选「取消自动关机」。

**Q：config.json 能手动改吗？**
A：不建议。`config.json` 由程序自动管理，手动修改格式错误可能导致程序异常。所有配置都应通过程序界面修改。

**Q：托盘图标不显示或点不动？**
A：点任务栏 `^` 小箭头展开隐藏图标区，看图标是否在里面，可以拖到外面。如果图标还在但点了没反应（僵尸图标），打开任务管理器找到「Windows 资源管理器」→ 右键 → 重新启动。

## 注意事项

- 游戏必须窗口化运行，全屏无法识别 UID

- 一条龙执行期间不要操作鼠标键盘，会干扰自动化流程

- 首次使用建议只勾选一个账号试运行，确认流程正常后再批量执行

- 电脑不要开启休眠/睡眠，避免定时任务无法触发

- 不同账号的 BetterGI 一条龙配置可以是不同的，每个账号独立选择

- `tesseract-ocr` 文件夹必须和 exe 在同一目录

- `config.json` 请勿手动编辑，所有设置通过程序界面完成

- 「调度器配置组」是 UID 不匹配时才触发的兜底机制，不是必填项，普通用户无需关心

- 要完全退出程序请右键托盘图标选「退出」，点 X 只是最小化到托盘
