原神多账号辅助工具 v5.4
==========================

概述
----
本工具可以自动切换多个原神账号，逐号调用 BetterGI 完成每日委托、清体力等自动化操作。支持直接启动 / 胡桃启动两种方式，内置定时计划，勾选开机自启后可以做到无人值守。

必备依赖：
- BetterGI（执行游戏内自动化）
- 原神游戏（窗口化运行）
- 胡桃工具箱（仅切号功能需要，可选）
- Tesseract OCR（压缩包已自带，无需单独安装）


第一步：安装依赖
----------------

1. BetterGI
BetterGI 是真正执行游戏内自动化操作的软件（战斗、委托、领取奖励等）。
- 下载地址：https://github.com/babalae/better-genshin-impact
- 下载最新 Release 的压缩包，解压到任意目录（不需要安装）。
- 打开 BetterGI，在「自动任务」→「一条龙」页面配置好你的任务流程。
  （不会配去 B 站搜 "BetterGI 一条龙教程"）

2. 原神
确保原神已安装，且启动方式为窗口化：
设置 → 图像 → 显示模式 → 窗口化。

3. 胡桃工具箱（可选）
如果你需要胡桃切号功能：
- 下载地址：https://github.com/DGP-Studio/Snap.Hutao
- MSIX 版（Microsoft Store 安装）需在设置中填写 AppID；普通解压版无需 AppID。
判断是否 MSIX 版：右键胡桃快捷方式 → 属性，路径含 WindowsApps 即为 MSIX 版。
在 PowerShell 中执行 Get-AppxPackage *Hutao* 获取 AppID。

4. Tesseract OCR（已内置，无需安装）
压缩包中已自带 tesseract-ocr 文件夹，包含中英文语言包。
首次使用时在设置里将「Tesseract OCR 目录」指向该文件夹即可。


第二步：文件部署
----------------
1. 将压缩包中所有文件解压到同一个文件夹（如 D:\GenshinTool）。
2. 文件夹路径不要含中文或空格。
3. 确保以下文件在同一目录下：
   · GenshinMultiAccountTool.exe —— 主程序
   · config.json —— 首次运行自动生成，不要手动编辑！
   · tesseract-ocr/ —— OCR 引擎（必须和 exe 同级）

tesseract-ocr/ 文件夹必须和 exe 放在同一目录，否则 UID 识别会失败。


第三步：首次设置
----------------
双击 GenshinMultiAccountTool.exe 启动程序（窗口标题为 GenshinMultiAccountTool v5.4），
点击标题栏右侧的「设置」按钮。

设置项共 14 项：

1. BetterGI 可执行文件
   点「浏览」选择 BetterGI 目录下的 BetterGI.exe。
   不知道在哪？桌面找 BetterGI 快捷方式 → 右键 →「打开文件所在位置」即可。
   常见位置：C:\Program Files\BetterGI\BetterGI.exe 或 D:\BetterGI\BetterGI.exe。
   实在找不到就在文件夹里搜索 BetterGI.exe。

2. BetterGI 配置文件
   点「浏览」选择 BetterGI 的 User\config.json。
   找到 BetterGI 文件夹后打开 User 子文件夹，选中 config.json。
   如果没看到 User 文件夹 → 打开 BetterGI.exe 一次（不进游戏也行），关掉后自动生成。

3. 原神可执行文件
   点「浏览」选择 YuanShen.exe（国际服选 GenshinImpact.exe）。
   桌面找原神快捷方式 → 右键 →「打开文件所在位置」。
   如果指向的是启动器，进去还有 Genshin Impact Game 子文件夹，点进去找。
   常见位置：C:\Program Files\Genshin Impact\Genshin Impact Game\YuanShen.exe。

4. 原神进程名
   默认 YuanShen.exe，国际服改 GenshinImpact.exe。写错检测不到游戏。
   国服/国际服判断：国内手机号登录 = 国服；邮箱和国外账号登录 = 国际服。

5. 胡桃工具箱路径（可选）
   仅胡桃切号时需要。选择 Snap.Hutao.exe。
   桌面找胡桃快捷方式 → 右键 →「打开文件所在位置」即可定位。
   常见位置：C:\Users\你的用户名\AppData\Local\Hutao\Snap.Hutao.exe。

6. 胡桃 AppID（可选）
   仅 MSIX 版胡桃需要。普通解压版留空。

7. 一条龙超时时间
   单个账号最大等待时间（秒），默认 7200（2 小时）。
   账号多或任务重的可以改大，超时未完成会跳过当前号。

8. 全局停止快捷键
   点「点击设置」后按下键盘组合键（如 F8、Ctrl+Shift+Q），支持修饰键 + 单键。
   一条龙执行中按下此键可强制停止。

9. 全局暂停/继续快捷键
   默认 Ctrl+Shift+P。一条龙执行中按下可暂停，再按继续。

10. 启动时自动最小化
    勾选后程序启动时自动缩到系统托盘。配合定时器的「一键启动全部」使用效果最佳。

11. 关闭窗口时最小化到系统托盘
    勾选后点 X 不退出，缩到右下角托盘图标。完全退出需右键托盘选「退出」。

12. 所有任务完成后自动关机
    勾选后全部账号执行完毕弹出 60 秒倒计时自动关机。倒计时期间可取消。

13. Tesseract OCR 目录
    选择解压后自带的 tesseract-ocr 文件夹（和 exe 同级）。
    自己装的常见位置：C:\Program Files\Tesseract-OCR，找不到搜索 tesseract.exe。

14. UID 识别方式
    ① Tesseract OCR（推荐）—— 需配好 OCR 目录，不依赖 BetterGI，但必须窗口化且画面不能遮挡。
    ② BetterGI 调度组（更准）—— BetterGI 内部直接读 UID，不需 Tesseract。
    选这个后在下拉框里选一个 BetterGI 调度组。

填写完毕后点「保存」。


第四步：添加账号
----------------
回到主界面，左侧面板底部点「+ 添加账号」，弹出账号编辑窗口。

字段说明：
- 账号名称：任意起名，如 "大号""小号""亚服号"
- 启动方式：直接启动(BetterGI)（推荐）或 胡桃启动（需胡桃工具箱做切号）
- BetterGI 一条龙配置：下拉选择，为空说明 BetterGI 未配一条龙任务
- 调度器配置组：UID 不匹配时执行，普通用户留空即可
- 胡桃中的账号名称：仅胡桃启动需要，须与胡桃中账号名完全一致
- 游戏UD：9 位数字，验证是否登入正确账号
- 任务完成后关闭软件：关闭游戏 / 关闭BetterGI / 关闭胡桃，默认全勾

其他操作：
- 点击行内「编辑」修改账号信息
- 选中账号后点「删除」移除
- 拖拽排序调整执行顺序（顺序即执行顺序）
- 底部「全选」一键勾选/取消全部


第五步：开始执行
----------------
启动前确认：
- 原神、BetterGI 均未运行（程序会自动启动它们）
- 游戏已设为窗口化模式
- 一条龙执行期间不要动鼠标键盘

点击右下角「▶ 开始」按钮。

按钮状态变化：
  开始前：开始(蓝) · 暂停(灰) · 停止(灰)
  运行中：开始(灰) · 暂停(橙) · 停止(红)
  暂停时：开始(灰) · 继续(绿) · 停止(红)
  停止后：开始(蓝) · 暂停(灰) · 停止(灰)

执行流程（逐号自动完成）：
1. 启动原神 → 2. 识别 UID 验证账号 → 3. UID 不匹配时执行调度器配置组 →
4. 启动 BetterGI 执行一条龙 → 5. 监控日志等待完成 →
6. 等 10 秒收尾 → 7. 关闭游戏/BetterGI/胡桃 → 8. 切下个号重复

若开启「自动关机」，全部完成后弹出红色「✕ 取消关机」按钮 + 60s 倒计时，可点取消。
右侧日志区实时显示进度，底部进度条同步更新。


快捷键（热键）
-------------
| 快捷键         | 默认值           | 功能               |
|---------------|-----------------|-------------------|
| 全局停止       | Ctrl+Shift+Q    | 强制停止当前任务     |
| 全局暂停/继续  | Ctrl+Shift+P    | 暂停或继续当前任务   |

可在设置中自定义，支持单键（F8）或组合键。启动自动注册，关闭自动注销。
底部状态栏显示当前已注册快捷键。


系统托盘
--------
程序运行后出现在任务栏右下角托盘区（tooltip：GenshinMultiAccountTool v5.4）。

左键点击托盘图标：显示主窗口

右键点击托盘图标：
- 显示窗口 —— 打开主界面
- 开始任务 —— 同「▶ 开始」
- 暂停任务 —— 暂停当前一条龙
- 停止任务 —— 停止当前一条龙
- 取消自动关机 —— 仅在倒计时期间生效
- 定时器 —— 子菜单：
  · 每个定时任务独立启用/停用勾选项
  · 一键启动全部 / 一键关闭全部
  · 有任务启用时父菜单显示勾选标记
- 设置 —— 打开设置窗口
- 退出 —— 彻底关闭程序

注意：右键「退出」才是真正关闭；点 X 只是缩到托盘（前提是设置了第 11 项）。


定时计划
--------
点击标题栏「定时计划」打开管理窗口（与主窗口同款图标）。

【添加任务】
1. 执行模式：每天 / 每周 / 一次性 / 指定日期
2. 时间：快捷按钮（凌晨4点/早8点/午12点/晚8点）一键设定，也可手动选时/分
3. 日期/星期：根据模式显示对应选择器
4. 账号：勾选本次执行的账号
5. 调度组：可选共用 ScriptGroup
6. 点「添加任务」

【管理任务】
顶部栏：
- 一键启动全部 —— 一次性启动全部定时任务
- 停止定时器 —— 停止正在运行的定时器
- 开机自启 —— 电脑开机自动运行
- 完成自动关机 —— 任务结束自动关机

已设任务列表：每行右侧有独立启用/停用按钮，可单独开关某个任务。
顶部「全选」和「删除选中」可批量管理。状态栏显示当前定时器状态。


UID 识别详解
-----------
方式一：Tesseract OCR（默认）
  多阈值投票法识别游戏画面 UID。需 tesseract-ocr 与 exe 同级且设置中指向该目录。

方式二：BetterGI 调度组（更准确）
  利用 BetterGI 脚本读 UID，适合 OCR 不稳定时使用。
  配置：1. 脚本放 User/JsScript/ → 2. BetterGI 新建调度组加入脚本 →
        3. 程序设置选「BetterGI 调度组」并选该组名称


调度器切号说明
-------------
定时计划中的「调度组」用于执行 BetterGI ScriptGroup 脚本（如切号脚本）。
官方脚本仓库：https://github.com/babalae/bettergi-scripts-list
推荐切号脚本：repo/js/SwitchAccountMultipleMode


常见问题
--------
Q：中途怎么停下来？
A：点「停止」按钮 / 按全局停止快捷键 / 右键托盘图标选「停止任务」。

Q：怎么临时暂停？
A：点「暂停」或按 Ctrl+Shift+P，再按一次继续。

Q：提示"未检测到 UID"？
A：检查 OCR 目录路径 → 确认窗口化 → 画面无遮挡 → tesseract-ocr 须与 exe 同级。
   还不行就切到 BetterGI 调度组方式。

Q：BetterGI 启动了但游戏里没反应？
A：检查一条龙任务已配置 → 确认 BetterGI 配置文件路径 → 点击游戏窗口使其置顶。

Q：账号切换失败？
A：确认配置文件路径 → 胡桃启动检查路径和账号名 → 检查 UID 填写。

Q：程序能打开多个吗？
A：不能。单实例锁定，重复启动提示「程序已在运行中」。找不到窗口右键托盘「显示窗口」。

Q：窗口关闭后怎么找回来？
A：缩到托盘了，双击托盘图标或右键「显示窗口」。右键「退出」才真关闭。

Q：进度条不走？
A：进度条显示 0/勾选数。卡住时检查 BetterGI 日志或调大超时时间。

Q：定时器到时间不执行？
A：确认状态「运行中」→ 电脑不能休眠（电源设"永不"）→ 有任务执行时会跳过新触发 →
   先手动点「一键启动全部」试一次。

Q：自动关机怎么取消？
A：主界面红色「✕ 取消关机」按钮 / 右键托盘「取消自动关机」。

Q：config.json 能手动改吗？
A：不建议。所有配置通过程序界面修改。

Q：托盘图标不显示或点不动？
A：点 ^ 展开隐藏图标区 → 拖到外面。僵尸图标：任务管理器 → Windows 资源管理器 → 重新启动。


注意事项
--------
- 游戏必须窗口化，全屏无法识别 UID
- 一条龙执行期间不要动鼠标键盘
- 首次使用只勾一个号试跑
- 电脑不要休眠/睡眠
- 不同账号可设不同一条龙配置
- tesseract-ocr 必须和 exe 同级
- config.json 勿手动编辑
- 调度器配置组非必填，普通用户无需关心
- 完全退出：右键托盘「退出」，点 X 只是最小化
